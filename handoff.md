## Реалізовані фічі

### Режими гри (6 шт)

Стартовий екран має дві групи кнопок:

**Переклад:**
- `🇺🇦 → 🇺🇸` (ua-en) — показ UA → ввод EN
- `🇺🇸 → 🇺🇦` (en-ua) — показ EN → ввод UA

**English only:**
- `📖 Def` (en-def) — показ визначення → ввод слова (~402 слова)
- `🔁 Syn` (en-syn) — показ слова → ввод синоніма (~164 слова)
- `↔ Ant` (en-ant) — показ слова → ввод антоніма (~120 слів)
- `📝 Cloze` (en-cloze) — речення з `____` → заповнити слово (~1392 слова)

Кожен режим будує свій пул слів на старті гри (фільтрує по доступності даних).

### Гра
- 60 секунд на раунд (звичайна гра)
- При помилці — correction-панель на 3.5 сек з правильною відповіддю, прикладом, TTS
- "Останнє слово" — після завершення таймера ще одна спроба на поточне слово
- TTS через Web Speech API (preferred female US English voice)

### Review-режим (повторення помилок)
- Запускається з end-екрану ("Повторити помилки") або зі стартового ("📒 Помилки за сьогодні")
- Слова з помилок поточного дня (з localStorage)
- **Правило освоєння:** 3 правильні відповіді на слово
- **TAINTED-логіка** (важливо!): якщо в review-сесії на слові була бодай **1 помилка/пропуск**, воно НЕ видаляється з "помилок за день" навіть після 3 правильних. Має чисто пройти.
- 60-сек таймер триває як страховка; після 0 — overtime з `+N` червоним
- Закінчується коли всі слова освоєні або користувач тисне "Закінчити"

### Hover translations у реченнях
- Кожне слово в EN-реченні обгорнуте в `<span class="hw">`
- Наведення → переклад (тултіп через `#hw-tooltip` div, JS-позиціонування з viewport-clamping)
- Клік → TTS того слова
- Стоп-слова (be/is/the/of/...) пропускаються
- Слова без перекладу теж обгортаються (для TTS)
- Покриття: ~99.8% слів у EXAMPLES мають переклад

### "to" prefix для полісемічних дієслів
25 noun/verb пар використовують "to X" для verb-форми:
`to address, to answer, to bear, to book, to clean, to close, to fall, to fly, to last, to lead, to lie, to like, to love, to mean, to open, to play, to point, to report, to ring, to season, to tear, to type, to waste, to watch, to wave`

- `isCorrect` приймає і "book", і "to book" (нормалізація)
- `EN_TO_UA` мапить і "to book", і "book" → обидва меньшого + більшого значення
- TTS озвучує лише "book" (без "to") для коротких verb-форм

### Рейтинги (Leaderboard)
Три вкладки збоку:
1. 🏆 Найкращі ігри — топ-10 окремих ігор
2. 📅 Найкращі дні — топ-10 днів (сума за день)
3. 🔥 Серії днів підряд — топ-10 streaks

Назви режимів у рейтингу: `UA→US`, `US→UA`, `Definition`, `Synonym`, `Antonym`, `Cloze`.

Топ-3 з золото/срібло/бронза. Сьогоднішня дата виділена.

### Гарячі клавіші
- **Enter** — submit / advance correction / start / restart
- **Esc** — end game OR close end-screen (нове)
- **Пробіл** — speak word / start review
- **Tab/Shift+Tab** — cycle leaderboard tabs
- **LeftShift** — cycle modes уперед (зліва→направо)
- **RightShift** — cycle modes назад (справа→наліво)

## Архітектура коду

### Дані (всі в index.html)
- `WORDS` — масив `{en, ua}` (1432 entries, ~1393 unique EN words)
- `TRANS` — IPA транскрипції (зберігається, не показується)
- `EXAMPLES` — EN речення-приклади (1433 keys)
- `EXAMPLES_UA` — UA переклади речень
- `DEFS` — англомовні визначення (476 entries) для Def-режиму
- `SYNS` — синоніми (187 entries) для Syn-режиму
- `ANTS` — антоніми (143 entries) для Ant-режиму

### Хелпери
- `getEn(w)`, `getUa(w)` — нормалізують string|array → array
- `isCorrect(input)` — валідація з "to" нормалізацією
- `lookupExample(table, enWord, uaWord)` — спершу шукає специфічне значення `"word:meaning"`
- `wrapSentence(sentence)` — обгортає слова в `.hw` спани з data-trans
- `stemEn(word)` — стемінг + IRREGULAR форми
- `speak(word)` — TTS, для "to X" speak'ить лише "X"
- `EN_TO_UA` — Map, reverse-map з urgent expansion на bare-форми

### localStorage ключі (НЕ перейменовувати без міграції!)
- `oxford_scores_v1` — топ-10 ігор
- `oxford_days_v1` — щоденна агрегація `{date: {correct, wrong, skipped, games, wordCount}}`
- `oxford_today_best_v1` — найкраща гра сьогодні
- `oxford_latest_v1` — остання гра
- `oxford_day_mistakes_v1` — помилки по днях `{date: [wordIndices]}`

### Стан гри (in-memory)
- `mode` — поточний з 6 режимів
- `reviewRemaining` — Set індексів, що ще потрібно освоїти в поточному review
- `reviewCorrectCounts` — `{idx: count}` правильних відповідей у поточному review
- `reviewTaintedIndices` — Set індексів з помилкою/пропуском у поточному review (НЕ видаляти з помилок дня навіть після 3 правильних)

### Візуальна система
- `#ffd166` — жовтий, is-new (highlight)
- `#87b4ff` — синій, "остання гра"
- `#ff6b6b` — червоний (помилки, overtime таймер)
- Top-3 рейтингу: золото/срібло/бронза тіні

## Що залишилось / можливі покращення

- **Варіант 2 для "to"** — поширити "to" на ВСІ дієслова (не тільки полісемічні). ~400 слів. Користувач відмовився, тільки полісемічні. Якщо знов спитає — дивись історію.
- **DEFS_UA** — українські переклади визначень. Зараз їх немає, hover на словах визначення дає per-word translation.
- **fetch-definitions.js** — Node.js скрипт для розширення DEFS/SYNS/ANTS через Free Dictionary API. У cloud-середовищі API заблоковане ("Host not in allowlist") → треба запускати локально на машині користувача.
- **Полісемія слів** — деякі слова мають більше значень, ніж записано. Користувач помічає по ходу гри і ми додаємо точково:
  - `area` тепер: площа/район/територія
  - `power` тепер: сила/влада
  - можливо ще будуть
